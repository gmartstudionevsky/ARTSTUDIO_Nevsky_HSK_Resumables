# Database (PostgreSQL) + Prisma

## Запуск локальной базы

```bash
npm run db:up
```

## Остановка базы

```bash
npm run db:down
```

## Полный сброс базы (с удалением volume)

```bash
npm run db:reset
```

## Миграции

```bash
npm run prisma:migrate
```

## Seed первого администратора

```bash
npm run seed
```

## Prisma Studio

```bash
npm run prisma:studio
```
