# Архитектурная формализация ядра (R1)

Этот каталог содержит архитектурный слой канона vNext, который связывает продуктовую спецификацию с последующими кодовыми этапами.

## Документы

- [domain-model.md](./domain-model.md) — карта сущностей и архитектурных границ.
- [write-flow-map.md](./write-flow-map.md) — классы и шаблон канонических write-flow.
- [read-model-map.md](./read-model-map.md) — классы и обязательные базовые проекции.
- [invariants.md](./invariants.md) — карта инвариантов и контуры защиты.
- [recovery-model.md](./recovery-model.md) — rollback/reset/re-sync и уровни применения.
- [movements-hub.md](./movements-hub.md) — архитектурно-прикладная карта хаба «Движения».
- [terminology-compatibility-r2.1.md](./terminology-compatibility-r2.1.md) — кодовая карта соответствия legacy-терминов и канонической модели R2.1.

## Место в дорожной карте

- R0 — документальный канон (завершён).
- R1 — архитектурная формализация ядра (завершена данным пакетом).
- R2.1 — первая кодовая волна канонической предметной модели (выполнена, compatibility-first).
- Далее — R2.2/R2.3: аналитические оси и усиление инвариантов поверх канонического слоя.
