# ADR-003: Нативная синхронность пространства

## Контекст

Нужно исключить фрагментарный подход, при котором синхронность ограничивается импортом или отдельными экранами.

## Решение

- Нативная синхронность принимается как сквозное свойство всей системы.
- Охват: канонические сущности, управляющие сущности и производные представления.
- Уровни синхронности:
  - сущностная;
  - проекционная;
  - управляемая.
- Центральное ядро: **Позиция учёта** и **Движение**.

## Последствия

- Все последующие архитектурные решения проверяются на совместимость с общей синхронностью.
- Исключается появление «локальных истин» в подсистемах.
